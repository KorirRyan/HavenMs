## How to Run This Project Locally
1. Download or clone this GitHub repository.
2. Move the project folder into your XAMPP directory: `C:/xampp/htdocs/`.
3. Open the **XAMPP Control Panel** and start both **Apache** and **MySQL**.
4. Go to `http://localhost/phpmyadmin/` in your browser.
5. Create a new database matching the project configuration.
6. Click the **Import** tab and upload the `.sql` file included in this repository.
7. Open `http://localhost/HavenMs/` to test the login.
